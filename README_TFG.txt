M&M Studio - Web para TFG ASIR

Esta web representa una empresa de asesoría de imagen digital y personal branding.
Incluye planes Normal, Premium y Exeltior basados en el proyecto de empresa original.

Archivos principales:
- inicio.html / index.html: página principal
- servicios.html: planes y precios
- asesoria-personal.html: servicio para personas
- marca-personal.html: servicio para empresas
- planes-asesoria.html: redes, analítica y reputación
- sobre-nosotros.html: resumen de empresa y parte técnica del TFG
- contacto.html y solicitar-plan.html: formularios simulados

Para probar con Docker:
1. Abrir terminal en esta carpeta
2. Ejecutar: docker compose up -d
3. Entrar en: http://localhost:8080

Mejoras técnicas que se pueden documentar en el TFG:
- Nginx como servidor web
- Docker Compose
- HTTPS con proxy inverso
- Backups automáticos
- Monitorización con Uptime Kuma
- Seguridad básica: firewall, permisos y cabeceras HTTP
